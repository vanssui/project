# VAN TRU FREE

Лёгкий недельный planner для задач семьи и личного ритма.  
Без сборщика, без фреймворков, без лишней тяжести: чистый `HTML`, `CSS` и `vanilla JavaScript`.

## О проекте

`VAN TRU FREE` — это не перегруженный task manager и не “корпоративный dashboard”.  
Это быстрый визуальный weekly tracker, в котором удобно:

- добавлять задачи на конкретные дни недели
- разделять их по категориям `Дом`, `Работа`, `Личное`
- отмечать срочные задачи
- быстро просматривать неделю целиком
- отправлять выполненное в архив

Главная идея проекта — сохранить ощущение лёгкости, скорости и визуального характера, не усложняя основной сценарий.

## Открыть

- Демо: [Открыть приложение](https://vanssui.github.io/project/)
- Локально: открой файл `index.html` из папки проекта
- Если браузер блокирует локальный запуск модулей, подними простой статический сервер из корня проекта:

```bash
python3 -m http.server 8080
```

После этого открой:

```text
http://localhost:8080
```

## Что умеет

- добавление, редактирование, удаление и дублирование задач
- недельная сетка с быстрым обзором по дням
- фильтры по категориям и срочности
- локальный поиск по задачам
- кнопка `Сегодня` для быстрого фокуса на текущем дне
- компактный режим отображения
- drag and drop задач между днями
- архив выполненных задач и его очистка
- импорт и экспорт JSON
- preview, edit, confirm и day modal
- toast-уведомления и мягкие feedback-состояния
- тёмная и светлая тема
- сохранение состояния в `localStorage`
- анимированный фон, часы и текущая дата

## Стек

- `HTML` — структура интерфейса
- `CSS` — визуальная система, темы, состояния и анимации
- `JavaScript ES Modules` — логика приложения без сборщика

## Архитектура

- `src/core` — бизнес-логика, вычисления, модели и селекторы
- `src/store` — состояние приложения и orchestration layer
- `src/services` — работа с `localStorage`, темой, фоном, импортом/экспортом и внешними действиями
- `src/ui` — DOM, рендер, карточки задач, модалки, события, toast
- `src/shared` — конфиг и общие утилиты
- `src/app/init.js` — точка входа

## Характер проекта

Проект специально сделан лёгким и прямым по сценарию:

- без тяжёлых зависимостей
- без backend-части
- без регистрации и аккаунтов
- без лишнего многослойного UI

Это deliberate tool: быстро открыл, быстро посмотрел неделю, быстро добавил задачу, быстро вернулся к делу.
